# 📘 วิธีใช้งานโปรเจกต์ — PF แนวข้อสอบ กพ

คู่มือสำหรับผู้พัฒนา (Dev) ว่าจะ **แก้ข้อสอบ / รันดูในเครื่อง / ขึ้นเว็บ (Vercel) / ทำไฟล์ APK** ยังไง

> ภาพรวม: เป็นเว็บแอป (PWA) ทำข้อสอบราชการ ก.พ. ภาค ก/ข/ค — **ข้อสอบทั้งหมดเก็บในไฟล์ `questions.xlsx`** แล้วระบบแปลงเป็นข้อมูลของเว็บ (`questions.js`) อัตโนมัติ · ใช้ได้ทั้งบนเว็บและเป็นแอป Android (.apk)

## 1) ⭐ แก้ / เพิ่มข้อสอบ (ทำใน Excel — สำคัญสุด)

ข้อสอบทั้งหมดอยู่ในไฟล์ **`questions.xlsx`** มี 3 ชีต:

| ชีต | เก็บอะไร | คอลัมน์ |
|---|---|---|
| **core** | ภาค ก | `subject` · `topic` · `year` · `predicted` · `passage` · `question` · `choice1`–`choice4` · `answer` · `explanation` · `freq` |
| **special** | ภาค ข / ภาค ค | `bank` · `topic` · `predicted` · `question` · `choice1`–`choice4` · `answer` · `explanation` · `freq` |
| **banks** | ข้อมูลคลังเฉพาะทาง | `bank` · `part` · `short` · `name` · `icon` |

**กติกาการกรอก:**
- `answer` = ใส่เลข **1–4** (ข้อที่ถูก เช่น 1 = choice1) — *(จะพิมพ์ A–D ก็ได้)*
- `predicted` = ใส่ **`yes`** ถ้าเป็นข้อเก็ง 2569 · ปล่อยว่างถ้าเป็นข้อสอบจริง
- `subject` (ชีต core) = `analytical` / `ethics` / `english` เท่านั้น
- `bank` (ชีต special) = ต้องตรงกับ id ในชีต **banks** เช่น `constitution`, `pdpa`, `local`, `procurement` …
- `freq` = อัตราการออก 1–5 (ปล่อยว่างได้ ระบบจะใส่ค่าเริ่มต้นให้)
- `passage` = ใส่เฉพาะโจทย์ที่มีบทอ่าน/ตาราง (ขึ้นบรรทัดใหม่ในเซลล์ได้ด้วย Alt+Enter)

**เพิ่มข้อสอบ** = เพิ่มแถวใหม่ในชีต core หรือ special · **แก้ข้อสอบ** = แก้ในแถวนั้น

เสร็จแล้วเลือกทำอย่างใดอย่างหนึ่ง:
```bash
npm run data        # อ่าน Excel → สร้าง questions.js + dist ใหม่ (ดูในเครื่องก่อน)
```
หรือ commit + push ขึ้น GitHub ได้เลย → **Vercel จะอ่าน Excel สร้างข้อมูลใหม่ตอน deploy ให้อัตโนมัติ**

> ⚠️ **อย่าแก้ `questions.js` ตรง ๆ** — ไฟล์นี้ถูกสร้างจาก `questions.xlsx` (แก้ไปก็จะโดนเขียนทับ)
> 🛡️ ถ้า Excel เสียหาย/กรอกผิดจนข้อมูลน้อยผิดปกติ ระบบจะ **ไม่เขียนทับข้อมูลเดิม** (กันข้อมูลหายตอน deploy)

## 3) 🚀 ขึ้นเว็บ (Deploy บน Vercel)

โปรเจกต์เชื่อมกับ GitHub: **github.com/aminadmin16/pf-exam** และตั้งค่า Vercel ไว้ใน `vercel.json` แล้ว
แค่ push โค้ดขึ้น GitHub → **Vercel จะ build + deploy ให้เองอัตโนมัติ**

```bash
git add -A
git commit -m "อัปเดตข้อสอบ / แก้เว็บ"
git push
```

หลัง deploy จะได้ 3 ลิงก์:
| ลิงก์ | คือ |
|---|---|
| `เว็บของคุณ.vercel.app` | ตัวเว็บแอป (เล่นได้เลย) |
| `…/landing` | หน้าดาวน์โหลดแอป (ปุ่มไอคอน + วิธีติดตั้ง) |
| `…/landing/PFExam.apk` | ไฟล์ APK โหลดตรง |

> เว็บใช้ **cache-first** — โหลดครั้งแรกครั้งเดียวแล้วเก็บไว้ในเครื่องผู้ใช้ เปิดครั้งต่อไปเร็วมากและใช้ host น้อยสุด

## 4) 📱 ทำไฟล์ APK (Android — เซ็นใบรับรองแล้ว)

**ต้องมี:** JDK 21 (`D:\jdk21\jdk-21.0.11+10`) + Android SDK · และไฟล์ใบเซ็น `android/app/kp-release.jks` (เก็บไว้ในเครื่อง)

**คำสั่งเดียวจบ (PowerShell):**
```powershell
$env:JAVA_HOME = "D:\jdk21\jdk-21.0.11+10"
npm run data                 # ถ้าเพิ่งแก้ข้อสอบใน Excel
npm run apk:release          # build เว็บ + คัดลอกเข้า android + build APK เซ็นแล้ว
Copy-Item "android\app\build\outputs\apk\release\app-release.apk" "landing\PFExam.apk" -Force
```
ได้ไฟล์ **`android\app\build\outputs\apk\release\app-release.apk`** (เซ็นด้วยใบรับรอง CN=PF Examination, ~3.4 MB)

> 🔑 **เก็บ `android/app/kp-release.jks` ไว้ให้ดี** (ไม่ได้ขึ้น Git) — ต้องใช้ใบเดิมตอน build เวอร์ชันอัปเดต ไม่งั้นผู้ใช้จะอัปเดตทับไม่ได้
> ℹ️ ตอนติดตั้งถ้า Google Play Protect เตือน “ไม่รู้จักนักพัฒนา” ให้กด **“ยืนยันว่าจะติดตั้ง”** (อย่ากด “รับทราบ”) → ปลอดภัย เป็นเพราะแอปยังใหม่ Google เลยยังไม่รู้จัก

## 5) 🧰 รวมคำสั่ง (npm scripts)

| คำสั่ง | ทำอะไร |
|---|---|
| `npm run data` | อ่าน `questions.xlsx` → สร้าง `questions.js` + `dist/` ใหม่ |
| `npm run data:export` | ส่งออก `questions.js` ปัจจุบัน → `questions.xlsx` (ใช้ตอนอยาก seed Excel ใหม่) |
| `npm run build:web` | สร้างโฟลเดอร์ `dist/` (เว็บพร้อมใช้) จากไฟล์ต้นทาง |
| `npm run zip:web` | build + แพ็กเป็น `kp-exam-web.zip` (เผื่ออยาก host แบบ static) |
| `npm run apk:release` | build เว็บ + cap copy + build APK (เซ็น release) |
| `npm run apk:debug` | build APK แบบ debug (ทดสอบ) |
| `npm run sync` | build เว็บ + คัดลอกเข้าโปรเจกต์ Android |

## 7) ❓ ปัญหาที่พบบ่อย

- **แก้ Excel แล้วเว็บไม่เปลี่ยน?** → ลืมรัน `npm run data` (ในเครื่อง) หรือลืม push (บน Vercel)
- **`npm run data` ขึ้น “ข้อมูลน้อยผิดปกติ”** → Excel กรอกผิด/ลบข้อมูลเยอะ ระบบเลยไม่เขียนทับ ให้เช็กชีต core/special/banks
- **ข้อสอบบางข้อหาย** → ดู log ตอนรัน `npm run data` จะบอกว่าแถวไหนผิด (เช่น answer ไม่ใช่ 1–4, ตัวเลือกซ้ำ, bank ไม่รู้จัก)
- **อยากเปลี่ยนสีธีม** → แก้ตัวแปรใน `:root { … }` ของ `styles.css`
- **ข้อสอบเป็นต้นฉบับ** เรียบเรียงขึ้นเองในแนวข้อสอบจริง ไม่ได้คัดลอกข้อสอบจริงของ ก.พ.
