# PF แนวข้อสอบ กพ

เว็บแอป (PWA) + แอป Android สำหรับฝึกทำ **แนวข้อสอบราชการ ก.พ. ภาค ก / ข / ค** พร้อมเฉลยละเอียด วิเคราะห์จุดอ่อน และข้อเก็งปี 2569 — **ฟรี ไม่มีโฆษณา**

> 📚 **คลังข้อสอบปัจจุบัน ~1,288 ข้อ** · ภาค ก (วิเคราะห์/จริยธรรม/อังกฤษ) · ภาค ข 12+ ฉบับกฎหมาย · ภาค ค (SJT + สัมภาษณ์) · สอบตามตำแหน่ง · แนวข้อสอบจำลองแยกตามปี 2565–2569 · ข้อเก็ง 2569
>
> ⚠️ **ข้อชี้แจง:** ก.พ. **ไม่เคยเผยแพร่ข้อสอบจริง**ต่อสาธารณะ ข้อสอบทั้งหมดในแอป (รวมถึงที่แยกตามปี) เป็น **แนวข้อสอบจำลองที่เรียบเรียงขึ้นใหม่** ตามผังสอบและแนวที่ออกบ่อยของปีนั้น ไม่ใช่ข้อสอบทางการของ ก.พ.

## 🚀 เริ่มต้น

```bash
npm install
npm run data                 # สร้างข้อมูลข้อสอบจาก questions.xlsx
python -m http.server 5500   # เปิด http://localhost:5500
```

📖 **คู่มือใช้งานละเอียด (แก้ข้อสอบ / deploy / ทำ APK):** ดูที่ [`วิธีใช้งาน.md`](./วิธีใช้งาน.md)

## ✨ ฟีเจอร์เด่น

- 📝 ข้อสอบจำลองเสมือนจริง 100 ข้อ (เรียงตามผังสอบของ ก.พ.) · ฝึกรายวิชา (เลือกจำนวนข้อ) · ฝึกเร็ว · แนวข้อสอบแยกตามปี
- 🎯 สอบตามตำแหน่ง (28 ตำแหน่ง 6 สายงาน) — จัดชุด ภาค ข + ภาค ค ให้
- 💡 เฉลยละเอียดทุกข้อ + **ปุ่มคำใบ้** ทุกข้อ + ⭐ ดาวอัตราการออก + 🔮 ป้ายข้อเก็ง
- 🔎 วิเคราะห์จุดแข็ง–จุดอ่อนหลังสอบ (กราฟวิ่ง) + ปุ่ม "ทดสอบเรื่องที่ยังควรเสริม"
- 📖 แท็บ "อ่านก่อนสอบ" — สรุป + สูตร + ทริค + ผังข้อสอบ
- 🎨 3 ธีม (สว่าง/มืด/อ่าน) · responsive ทุกขนาดจอ · ออฟไลน์ได้ (cache-first)

## 🛠️ เทคโนโลยี

- **เว็บ:** Vanilla HTML/CSS/JS (ไม่ใช้ framework) · PWA (service worker cache-first)
- **ข้อมูลข้อสอบ:** เก็บใน `questions.xlsx` → แปลงเป็น `questions.js` อัตโนมัติ (SheetJS)
- **Android:** Capacitor → ไฟล์ `.apk` (เซ็น release)
- **Deploy:** Vercel (อ่าน Excel → build ตอน deploy)

## 📁 ไฟล์สำคัญ

| ไฟล์ | คือ |
|---|---|
| `questions.xlsx` | ⭐ ข้อสอบทั้งหมด (แก้ที่นี่) |

| `study-notes.js` | เนื้อหา "อ่านก่อนสอบ" |
| `landing/` | หน้าดาวน์โหลดแอป |
| `_batches/xlsx-to-questions.js` | แปลง Excel → questions.js |
| `vercel.json` | ตั้งค่า deploy |
