# แอปแนวข้อสอบ ก.พ. ภาค ก (พร้อมเฉลย + สรุปผลคะแนน)

แอปพลิเคชันฝึกทำ **แนวข้อสอบ ก.พ. ภาค ก** ครบทั้ง 3 วิชา พร้อมเฉลยละเอียด (วิธีคิด)
และหน้าสรุปผลคะแนนรูปแบบเดียวกับรายงานผลของสำนักงาน ก.พ.

> 🗂️ **คลังข้อสอบปัจจุบัน: 855 ข้อ — ครบทั้ง ภาค ก / ข / ค**
> - **ภาค ก (ความรู้ความสามารถทั่วไป) 518 ข้อ** (คิดวิเคราะห์ 250 · กฎหมาย/ข้าราชการที่ดี 137 · ภาษาอังกฤษ 131) คละปี **2565–2569** — รวม **ข้อสอบเก็ง 2569 จำนวน 98 ข้อ**
> - **ภาค ข (ความรู้เฉพาะตำแหน่ง) 235 ข้อ** 12 ฉบับ: รัฐธรรมนูญ 2560 · ข้าราชการพลเรือน 2551 · พนักงานราชการ 2547 · ข้อมูลข่าวสารฯ 2540 · PDPA 2562 · จัดซื้อจัดจ้าง 2560 · งานสารบรรณ 2526 · วินัยการเงินการคลัง 2561 · ยุทธศาสตร์ชาติ 20 ปี · พ.ร.บ.คอมพิวเตอร์ · ปกครองท้องถิ่น · การศึกษาแห่งชาติ 2542
> - **ภาค ค (ความเหมาะสมกับตำแหน่ง) 106 ข้อ**: ข้อสอบสถานการณ์ SJT (บริการ/ทีมงาน, จริยธรรม/แก้ปัญหา) + เตรียมสอบสัมภาษณ์
> - **🎯 สอบตามตำแหน่ง** 9 ตำแหน่ง (จัดชุด ก+ข+ค ให้อัตโนมัติ) · ทุกข้อมี **ดาวอัตราการออก ★** · สุ่มสลับข้อใหม่ทุกครั้ง

> ⚠️ ข้อสอบทั้งหมดในแอปนี้ **เรียบเรียงขึ้นใหม่เอง** ให้อยู่ใน “แนว” เดียวกับข้อสอบจริง
> ปี 2565–2568 (ย้อนหลัง 4 ปี) ไม่ได้คัดลอกข้อสอบจริงของ ก.พ. (ซึ่งเป็นความลับของทางราชการ)
> จึงใช้ฝึกฝนได้อย่างสบายใจ

## ➕ วิธีเพิ่มข้อสอบเอง

เปิดไฟล์ `questions.js` แล้วเพิ่มอ็อบเจกต์ใหม่เข้าไปในอาเรย์ `window.QUESTIONS` ตามรูปแบบนี้:

```js
{
  id: 241,                      // เลขลำดับ (ห้ามซ้ำ)
  subject: "analytical",        // analytical | ethics | english
  topic: "อนุกรมตัวเลข",         // หมวดย่อย (แสดงเป็นป้ายกำกับ)
  year: 2567,                   // ปี พ.ศ. ของแนวข้อสอบ (2565–2568)
  passage: "บทความ/บทอ่าน...",   // (ใส่หรือไม่ก็ได้) ใช้กับโจทย์ที่มีบทอ่าน
  question: "โจทย์คำถาม...",
  choices: ["ตัวเลือก A", "ตัวเลือก B", "ตัวเลือก C", "ตัวเลือก D"],
  answer: 3,                    // ดัชนีคำตอบที่ถูก (เริ่มนับจาก 0 → 3 = ตัวเลือก D)
  explanation: "ตอบ ...\n\nวิธีคิด..."   // ใช้ \n เพื่อขึ้นบรรทัดใหม่
}
```

> 💡 `questions.js` ถูกสร้างรวมจากสคริปต์ `_batches/assemble.js` — ถ้าต้องการเพิ่มข้อสอบจำนวนมาก
> ให้เพิ่มไฟล์ `.json` ในโฟลเดอร์ `_batches/` แล้วรัน `node _batches/assemble.js` (จะรวม+ตรวจ+ตัดซ้ำ+ติดปีให้อัตโนมัติ)
> หรือจะแก้ไข `questions.js` ตรง ๆ ก็ได้ (เพียงอย่ารันสคริปต์ทับภายหลัง)

> ระบบจะ **สุ่มสลับลำดับข้อและตัวเลือก** ให้อัตโนมัติทุกครั้งที่ทำข้อสอบ
> และคำนวณคะแนนเต็มของแต่ละวิชาตามจำนวนข้อที่มีในคลังให้เอง

## 💝 ตั้งค่าช่องทางบริจาค (PromptPay QR)

แก้ไฟล์เดียว: **`donate-config.js`** เลือกวิธีใดวิธีหนึ่ง

**วิธีที่ 1 (ง่ายสุด) — ใส่เบอร์พร้อมเพย์ แล้วระบบสร้าง QR ให้เอง**
```js
window.DONATE = {
  enabled: true,
  promptpayId: "0812345678",   // ← เบอร์มือถือพร้อมเพย์ของคุณ (หรือเลขบัตร ปชช. 13 หลัก)
  payeeName: "ชื่อบัญชีของคุณ",
  ...
};
```
QR สร้างจากไลบรารีในเครื่อง (`vendor/qrcode-generator.js`) → **ใช้งานออฟไลน์ได้** และเลือกจำนวนเงิน ฿20/50/100 ได้

**วิธีที่ 2 — ใช้รูป QR จากแอปธนาคาร**
- เซฟรูป QR พร้อมเพย์ → วางที่ `assets/donate-qr.png`
- เว้น `promptpayId` ให้ว่างไว้

> ปุ่มบริจาคจะอยู่ที่ **ท้ายหน้าแรก** และ **หน้าสรุปผลหลังทำข้อสอบ** · ตั้ง `enabled: false` เพื่อซ่อนทั้งหมด

## 📱 ติดตั้งเป็นแอป (PWA) + วิธีทำไฟล์ APK

แอปนี้เป็น **PWA** ติดตั้งและใช้ออฟไลน์ได้ (มี `manifest.json`, `sw.js`, ไอคอนใน `icons/`)

**ติดตั้งเป็นแอป (ง่ายสุด ไม่ต้องมี APK):**
1. รันผ่านเซิร์ฟเวอร์ (`python -m http.server 5500`) หรือโฮสต์ออนไลน์
2. เปิดด้วย **Chrome บนมือถือ** → เมนู (⋮) → **“ติดตั้งแอป / เพิ่มลงในหน้าจอหลัก”**
   (บนคอม: กดไอคอนติดตั้งที่แถบที่อยู่)
→ ได้ไอคอนแอปบนมือถือ เปิดเต็มจอเหมือนแอปจริง

**ทำเป็นไฟล์ .apk จริง (เลือกทางใดทางหนึ่ง):**

🅰️ **PWABuilder (แนะนำ — ไม่ต้องเขียนโค้ด/ลงโปรแกรม)**
1. อัปโหลดโฟลเดอร์นี้ขึ้นโฮสต์ฟรี เช่น Netlify Drop / GitHub Pages → ได้ลิงก์ https
2. ไปที่ **https://www.pwabuilder.com** → ใส่ลิงก์ → **Package For Stores → Android** → ดาวน์โหลด `.apk`/`.aab` (เซ็นชื่อให้พร้อมติดตั้ง)

🅱️ **Capacitor — โปรเจกต์ Android ตั้งค่าไว้ให้แล้วในโฟลเดอร์ `android/`**
(appId: `com.kpexam.app` · appName: "ติวสอบ ก.พ." · เว็บฝังจาก `dist/`)

ติดตั้ง **Android Studio** (มาพร้อม JDK + Android SDK) แล้วเลือกวิธีใดวิธีหนึ่ง:

*วิธี GUI (ง่ายสุด):*
1. เปิด Android Studio → **Open** → เลือกโฟลเดอร์ `android`
2. รอ Gradle sync เสร็จ (ครั้งแรกโหลด SDK/Gradle สักครู่)
3. เมนู **Build → Build Bundle(s)/APK(s) → Build APK(s)**
4. ได้ไฟล์ `android\app\build\outputs\apk\debug\app-debug.apk` → ส่งเข้ามือถือ → ติดตั้ง

*วิธีคำสั่งเดียว (หลังลง Android Studio + ตั้ง ANDROID_HOME แล้ว):*
```bash
npm run apk:debug      # สร้าง dist + คัดลอกเข้า android + build APK ให้เลย
```

🔄 **แก้ข้อสอบ/โค้ดเว็บภายหลัง** → รัน `npm run sync` แล้ว build ใหม่ (อัปเดต dist เข้า Android อัตโนมัติ)

> เครื่องที่สร้างแอปนี้ยังไม่มี Java/Android SDK จึง build .apk ในเครื่องไม่ได้ — ใช้ 1 ใน 2 วิธีข้างบนได้เลย

## 🛠️ การปรับแต่ง

- **เปลี่ยนสีธีม** → แก้ตัวแปรสีในส่วน `:root { ... }` ของไฟล์ `styles.css`
- **เปลี่ยนเกณฑ์/คะแนนเต็มรายวิชา** → แก้ที่ `window.SUBJECTS` ในไฟล์ `questions.js`
- **เปลี่ยนจำนวนข้อของ “ฝึกเร็ว”** → แก้เลข `.slice(0, 10)` ในฟังก์ชัน `buildQuestionSet` ของ `app.js`
